🔗 Supabase Bağlantısı

Kullanıcı girişi için Supabase proje bilgilerini girin

Yapılandırılmamış

Supabase → Project Settings → API → Project URL

Supabase → Settings → API Keys → Legacy → anon public

Supabase → Settings → API Keys → Legacy → service_role → Reveal → Copy. Kullanıcı eklemek için gereklidir.

Nasıl kurulur?
1. supabase.com'a gidin, ücretsiz hesap açın
2. "New Project" ile proje oluşturun
3. Authentication → Email Auth açık olduğunu doğrulayın
4. Authentication → Email Confirm → kapatın (Confirm email: OFF)
5. Yukarıdaki URL ve Key'i kopyalayıp yapıştırın
