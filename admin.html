🔗 Supabase Bağlantısı

Kullanıcı girişi için Supabase proje bilgilerini girin

Yapılandırılmamış

Supabase → Project Settings → API → Project URL

Supabase → Project Settings → API → anon public

Supabase → Project Settings → API → service_role (secret) — Yalnızca admin panelinde saklanır, dışarıya açılmaz.

Nasıl kurulur?
1. supabase.com adresine gidin ve ücretsiz hesap açın
2. "New Project" ile yeni proje oluşturun
3. Authentication → Providers → Email → "Enable email" açık olduğunu doğrulayın
4. Authentication → Email → "Confirm email" → kapatın (OFF)
5. Yukarıdaki URL ve Key'i kopyalayıp yapıştırın
